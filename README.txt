Installation

1.- Instialize npm wit 'npm init -y'
2.- Install Testcafe with 'npm i testcafe'
3.- Install Testcafe Report with 'npm i testcafe-reporter-html'
4.- Install dotev with 'npm i dotenv'
5.- Install Newman with 'npm i newman'
6.- Install Newman Report with 'newman-reporter-htmlextra'
7.- Install k6 with 'npm i k6'
